# core.biz.pl
Website of project core

## Emulator Portfela Bitcoin Core

To jest internetowy emulator portfela Bitcoin Core, stworzony dla celów edukacyjnych. Symuluje podstawowe funkcje portfela Bitcoin bez prawdziwego połączenia z siecią Bitcoin.

### Funkcje

- Generowanie nowego portfela z losowym adresem
- Wyświetlanie salda
- Symulacja wysyłania BTC
- Symulacja odbioru BTC
- Historia transakcji

### Jak używać

1. Otwórz `index.html` w przeglądarce.
2. Kliknij "Generuj Nowy Portfel" aby stworzyć portfel.
3. Użyj sekcji "Wyślij BTC" aby symulować transakcje wychodzące.
4. Użyj sekcji "Odbierz BTC" aby symulować transakcje przychodzące.
5. Historia transakcji jest wyświetlana na dole.

### Uwagi

- To jest tylko symulacja. Nie używać prawdziwych kluczy prywatnych ani środków.
- Dane są przechowywane w localStorage przeglądarki.

### Troubleshooting

- Jeśli strona nie ładuje się, upewnij się, że wszystkie pliki (index.html, style.css, script.js) są w tym samym folderze.
- Jeśli dane nie zapisują się, sprawdź ustawienia prywatności przeglądarki dotyczące localStorage.
